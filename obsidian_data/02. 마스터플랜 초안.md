# 좀비파크 마스터플랜 초안
> 작성일: 2026-04-17 | 상태: 초안 (대화를 통해 계속 발전시킬 예정)

## 설계 원칙
1. **전체 청사진은 완성본을 그리되, 실행은 씨앗부터 심는다**
2. 처음부터 거액 투자 유치가 아닌, 작은 인프라부터 컨셉을 잡고 인지도를 쌓아 확장
3. 1단계에서 만든 것이 나중에 전체 파크에서 유기적 역할을 하도록 설계
4. 생존 거점이 하나씩 늘어나는 과정 자체가 아포칼립스 세계관과 일치

## 영흥도 방위별 배치

### 🛡️ 동쪽 (영흥대교/내리) — 방어 전선 + 1단계 서바이벌 캠프
- **영흥대교 = 유일한 진입로이자 유일한 방어 포인트**
- 바리케이드, 검문소, 감시초소 배치
- 방문객이 다리를 건너는 순간 "검문소 통과 → 생존 구역 진입" 경험
- **⭐ 1단계 착수 지점**: 최소 비용으로 서바이벌 훈련 캠프 운영 시작
  - 텐트/컨테이너 기반, 야외 생존 훈련 프로그램
  - 응급처치, 정수, 불 피우기, 바리케이드 구축 등
  - 주말 프로그램 → 정기 프로그램으로 확장

### 🏔️ 중앙 (국사봉 127.7m) — 지휘본부 / 정찰 거점
- 섬 최고봉에서 사방 조망 가능
- **자전거 페달 방식 케이블카**로 국사봉까지 이동 (전기 없이 구동되는 어트랙션)
- 통신 거점: 무전기 훈련, 신호 체계 구축 훈련
- 망루/감시탑 역할

### 🌾 서쪽 (장경리 해수욕장) — VIP존 / 장기 생존 구역
- **VIP 숙박형 서바이벌 체험 구역**
  - 1.5km 해변 + 100년 소나무숲 = 최적의 숙박 체험 환경
  - 세미 하드코어: 기본 편의시설 + 빡빡한 미션·훈련
  - 72시간 생존 시뮬레이션 프로그램
- **식량 인프라 — 곤충 사육 (니오타니 연계 가능성)**
  - 장경리에 이미 니오타니 곤충 사육관 존재 (운영자와 친분)
  - 마스터플랜상 역할: 아포칼립스 식량 3축 중 "단백질원" 담당
  - 평시: 곤충 체험관 / 세계관 내: 비상 단백질 공급 기지
  - 현재 상태: 정보 습득 단계 (협업 논의 전)

### ⚡ 남쪽 (외리) — 에너지 구역
- 영흥화력발전소가 이미 위치한 곳
- 파크 자체 에너지 시설: **전기가 끊긴 세상의 대안 에너지**
  - 태양광 패널 + 풍력 + 인력발전 (자전거 발전기 등)
  - "전기 없이 살아남기" 훈련 프로그램
- 아포칼립스 설정: 화력발전소는 연료 공급 끊기면 정지 → 대안 에너지가 핵심

### 🐚 남쪽 해안 (용담리) — 식량 채취 구역
- 갯벌 38.6km² (섬 면적의 1.6배)
- 바지락, 굴 등 해산물 채취 훈련
- 갯벌 생태 교육 + 비상 식량 확보 훈련
- 아포칼립스 식량 3축 중 "해산물" 담당

### 🏖️ 북쪽 (십리포) — 의료 거점 / 후방 기지
- 영흥대교에서 가장 먼 구역 → 방어 관점에서 안전한 후방
- 의료 시설, 약초원, 응급처치 고급 훈련소
- 빗물 집수 + 정수 메인 시설 (물 라이프라인)

| 해산물 | 용담리 갯벌 | 바지락, 굴, 조간대 생물 채취 | 80여 종 서식, 38.6km² 갯벌 |
| 작물 | 내륙 농경지 | 텃밭, 작물 재배 | 기존 농경지 다수 존재 |

## 점진적 확장 전략

### 1단계: 서바이벌 훈련 캠프 (최소 비용 시작)
- **위치**: 동쪽 영흥대교 인근
- **형태**: 텐트/컨테이너 기반 야외 캠프
- **프로그램**: 주말 서바이벌 체험 (응급처치, 정수, 불 피우기, 바리케이드)
- **목표**: 컨셉 검증 + 인지도 확보 + 운영 데이터 수집
- **수익**: 체험 프로그램 참가비 + 1등급 자격증 발급

### 2단계: 핵심 존 구축
- 서바이벌 캠프 확장 + 국사봉 정찰 거점 추가
- 자격증 등급 시스템 본격 도입

### 3단계: VIP존 + 식량 인프라
- 장경리 VIP 숙박 체험 오픈
- 곤충 사육/갯벌 채취 프로그램 정식 운영

### 4단계+: 풀 스케일 파크
- 전 구역 완성, 리조트급 시설
- 기업 단체 훈련 프로그램 (B2B)
- 연간패스/시즌 이벤트 운영

## 참고 레퍼런스
- **옹진소청초 해양과학기지**: 해상 자급자족 거점의 실제 사례. 담수화 설비, 태양광, 체류 시설 보유. 기술 레퍼런스 + 세계관 "해상 전초기지" 역할 가능
