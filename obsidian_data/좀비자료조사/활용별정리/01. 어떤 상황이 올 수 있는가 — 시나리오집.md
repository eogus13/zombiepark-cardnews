# 01. 어떤 상황이 올 수 있는가 — 시나리오집
> "좀비는 아직 없다. 하지만 올 수도 있다."
> 이 문서는 '어떤 형태로 올 수 있는가'를 상상하기 위한 시나리오집이다.
> 영화·신화·과학은 단서이자 참고일 뿐이다.

---

## 이 문서를 읽기 전에

우리가 좀비라고 부르는 건 하나의 정해진 괴물이 아니다.
**인간이 인간이 아닌 것으로 변하는 모든 상황**을 말한다.

영화에서는 이미 수십 가지 시나리오를 보여줬고,
과학은 실제로 생물의 행동을 조종하는 기생충과 곰팡이를 발견했고,
전 세계 신화는 수천 년 전부터 "죽은 자가 돌아온다"고 경고했다.

이것들을 종합하면 하나의 질문이 남는다:
> **"만약 내일 이런 일이 생긴다면, 어떤 형태로 올 것인가?"**

---

## 시나리오 1: 바이러스 팬데믹형
> "코로나는 기침을 시켰다. 다음 바이러스는 뭘 시킬까?"

### 가정
어떤 바이러스가 중추신경계를 공격하여 감염자의 **공격성을 극대화**하고 **이성을 마비**시킨다.
물림이나 체액으로 전파된다.

### 왜 가능한가?
- **광견병**이 이미 이것을 한다. 물림으로 전파, 공격성 증가, 물 공포증, 증상 발현 후 거의 100% 사망. 다만 잠복기가 길고 전파가 느릴 뿐이다.
- 만약 광견병의 공격성 + 독감의 전파력이 결합된 변이가 나온다면?
- 코로나19가 보여줬다 — 바이러스는 예고 없이, 전 세계를 동시에 친다.

### 이 시나리오에서 예상되는 상황
| 요소 | 예상 |
|------|------|
| 감염 속도 | 빠름 (수 분~수 시간) |
| 감염자 행동 | 극도의 공격성, 달리기, 물어뜯기 |
| 의식 유무 | 거의 없음 (본능만 남음) |
| 전파 방식 | 물림, 체액, 가능하면 비말 |
| 대응 핵심 | 격리, 보호복, 백신 개발 시간 벌기 |

### 참고한 단서들
- 광견병 바이러스의 실제 증상 → [03. 신화·과학 자료 참조]
- 영화 《28 Days Later》의 '레이지 바이러스' — 10초 감염이라는 극단적 가정
- 영화 《부산행》 — 화학물질 누출로 인한 감염, 수 분 내 전환
- 드라마 《해피니스》 — 약물 부작용이 바이러스로 전이되는 시나리오

## 시나리오 2: 곰팡이/기생충형
> "자연은 이미 좀비를 만들고 있다. 아직 인간 차례가 안 왔을 뿐이다."

### 가정
곰팡이 또는 기생충이 인간의 **신경계를 장악**하여 행동을 조종한다.
숙주는 살아있지만 자기 의지로 움직이지 않는다.

### 왜 가능한가?
- **오피오코르디셉스(동충하초)**: 지금 이 순간에도 개미를 좀비로 만들고 있다. 뇌를 침입하지 않고 근육을 직접 조종하여 개미를 원하는 장소로 이동시킨 뒤 죽인다.
- **톡소플라스마**: 전 세계 인구의 1/3이 감염되어 있다. 쥐의 뇌를 조작해서 고양이를 무서워하지 않게 만든다 — 즉 **이미 포유류의 행동을 바꾸는 기생충이 존재**한다.
- 이 두 가지가 인간에게 적응하려면 뭐가 필요한가? 진화는 시간문제다.

### 이 시나리오에서 예상되는 상황
| 요소 | 예상 |
|------|------|
| 감염 속도 | 느림 (수 일~수 주에 걸쳐 서서히) |
| 감염자 행동 | 초기엔 정상 → 점점 이상 행동 → 완전 장악 |
| 의식 유무 | 초기엔 있음 (자기가 변하고 있다는 걸 알 수도) |
| 전파 방식 | 포자 흡입, 접촉, 음식/물 오염 |
| 대응 핵심 | 마스크/방독면, 식수 관리, 조기 발견 |

### 참고한 단서들
- 실제 오피오코르디셉스 감염 메커니즘 → [03. 신화·과학 자료 참조]
- 게임 《The Last of Us》 — 코디셉스 변이체가 인간을 감염시키는 시나리오
  - 감염 단계: 러너 → 스토커 → 클리커 → 블로터 (단계별 변이)
  - 이건 픽션이지만, 실제 곰팡이도 단계별로 숙주를 변형시킴
- 드라마 《스위트홈》 — 내면의 욕망이 외형으로 발현되는 "괴물화"

## 시나리오 3: 화학/약물 사고형
> "인간이 만든 것이 인간을 덮친다."

### 가정
군사용 생화학 물질, 실험 약물, 또는 산업 폐기물이 유출되어
접촉하거나 흡입한 사람들의 **뇌 기능에 비가역적 손상**을 일으킨다.

### 왜 가능한가?
- 일본 731부대는 실제로 인간 대상 생물무기를 실험했다.
- 소련의 바이오프레파라트(Biopreparat)는 냉전기 세계 최대 생물무기 프로그램이었다.
- MK울트라 프로젝트에서 CIA는 약물로 인간의 정신을 조종하려 했다.
- 보팔 사고(1984) — 화학 공장 가스 누출로 한 밤에 수천 명 사망, 수십만 명 후유증

### 이 시나리오에서 예상되는 상황
| 요소 | 예상 |
|------|------|
| 발생 범위 | 국지적 (공장, 연구소, 군사 시설 주변) |
| 감염자 행동 | 예측 불가 (약물 종류에 따라 다름) |
| 의식 유무 | 부분적 (환각, 착란 상태) |
| 전파 방식 | 직접 노출 (공기, 물, 토양) |
| 대응 핵심 | 대피, 풍향 파악, 방독면, 오염 지역 격리 |

### 참고한 단서들
- 영화 《Resident Evil》시리즈 — 엄브렐라사의 T-바이러스 유출 사고
- 드라마 《Dark Hole》 — 싱크홀에서 나온 미지의 연기가 사람을 변이시킴
- 영화 《기묘한 가족》 — 제약회사 실험 폐기물이 좀비를 만듦
- 드라마 《해피니스》 — '넥스트' 약물 부작용이 통제 불능으로

## 시나리오 4: 프리온/신경퇴행형
> "천천히, 조용히, 치료법 없이."

### 가정
변형 프리온 단백질이 음식이나 환경을 통해 확산되어
감염자의 **뇌를 스펀지처럼 구멍 내고**, 행동 장애와 공격성을 유발한다.

### 왜 가능한가?
- **크로이츠펠트-야콥병(CJD)**: 실제 존재하는 프리온 질환. 치료법 없음. 진단 후 70%가 1년 내 사망.
- **광우병(BSE)**: 오염된 소고기를 먹고 사람이 변종 CJD에 감염된 실제 사례.
- **쿠루병**: 파푸아뉴기니의 포레 부족이 장례 의식으로 뇌를 먹어 집단 감염.
- 프리온은 바이러스가 아니다. **단백질**이다. 열에 강하고, 소독이 안 되고, 백신이 없다.

### 이 시나리오에서 예상되는 상황
| 요소 | 예상 |
|------|------|
| 감염 속도 | 매우 느림 (수 개월~수 년 잠복) |
| 감염자 행동 | 치매 → 불수의운동 → 공격성 → 식물 상태 |
| 의식 유무 | 점진적 소실 |
| 전파 방식 | 오염된 식품, 수혈, 수술 도구 |
| 대응 핵심 | 식품 안전, 조기 발견 시스템, 격리 |
| 가장 무서운 점 | **증상이 나타났을 때는 이미 수년간 퍼진 후** |

### 참고한 단서들
- CJD, BSE, 쿠루병의 실제 사례 → [03. 신화·과학 자료 참조]
- 이건 영화에서도 잘 다루지 않는 시나리오다. 그래서 더 무섭다.

## 시나리오 5: 불명/초자연형
> "과학으로 설명 안 되는 것이 오면?"

### 가정
원인을 모른다. 설명할 수 없다. 그냥 죽은 사람이 일어난다.
혹은 알 수 없는 힘에 의해 사람이 변한다.

### 왜 고려해야 하는가?
인류는 수천 년 동안 "죽은 자가 돌아온다"는 이야기를 해왔다.
문화권마다 독립적으로, 시대를 초월해서.

이게 전부 상상일까? 아니면 **인류가 경험한 무언가의 기록**일까?

| 문화 | 이름 | 시기 | 핵심 |
|------|------|------|------|
| 아이티 | 좀비(Zombie) | 17세기~ | 주술로 시체 부활. 실제 사례 존재(나르시스) |
| 중국 | 강시(殭屍) | 고대~ | 경직된 시체가 뛰어다님. 시신 운송 전통에서 유래 |
| 북유럽 | 드라우그르 | 중세~ | 무덤에서 일어난 전사. 초인적 힘 |
| 유럽 | 리버넌트 | 12세기~ | 전염병 시기 급증. 뱀파이어 전설과 연결 |
| 아랍 | 구울 | 고대~ | 묘지의 시체 포식자. 변신 능력 |
| 한국 | 귀신/야귀 | 고대~ | 원한으로 돌아온 죽은 자 |
| 일본 | 가샤도쿠로 | ~1000년 전 | 전사자들의 원혼이 합쳐진 거대 해골 |

### 이 시나리오의 대응 원칙
과학으로 설명 안 되는 상황이라도, **대응 원칙은 같다**:
- 위협을 피한다
- 안전한 곳으로 간다
- 생존에 필요한 것을 확보한다
- 함께 살아남는다

## 시나리오 6: 복합형 — 가장 현실적인 시나리오
> "하나가 아니라 여러 개가 동시에 터진다."

### 가정
팬데믹 + 사회 붕괴 + 인프라 마비가 동시에 일어난다.
좀비가 아니어도, **"좀비 상황"은 올 수 있다.**

### 생각해볼 것
- 코로나19 초기를 기억해보자. 마스크 사재기, 식량 매점매석, 국경 봉쇄.
- 거기에 전력망 붕괴, 통신 두절, 치안 공백이 더해진다면?
- 사람들이 패닉에 빠져 폭력적으로 변한다면?
- **"좀비"가 아니라 "좀비 같은 상황"은 이미 상상할 수 있다.**

### 실제 참고 사례
| 사례 | 연도 | 상황 |
|------|------|------|
| 카트리나 허리케인 | 2005 | 도시 기능 마비, 약탈, 치안 붕괴 |
| 후쿠시마 원전 사고 | 2011 | 방사능 대피, 격리, 정보 혼란 |
| 코로나19 | 2020 | 전 세계 동시 팬데믹, 사재기, 봉쇄 |
| 텍사스 대정전 | 2021 | 전력망 붕괴, 난방 두절, 식수 오염 |

### 이 시나리오에서의 핵심
좀비가 있든 없든, **준비해야 할 것은 같다**:
1. 물과 식량 확보
2. 안전한 거점 확보
3. 통신 수단 확보
4. 신뢰할 수 있는 사람들과 함께
5. 탈출 경로 확보
