### 4.1 전환 퍼널

```
Page View (1,234명 / 100%)
    ↓ segment_view (468명 / 38%)
세그먼트 선택 (고객층 탭 클릭)
    ↓ cta_click (247명 / 20%)

    ↓ form_start (156명 / 13%)

    ↓ form_submit (64명 / 5.2%)
폼 제출 (최종 전환) ⭐
```

**단계별 드롭율 분석**:
1. Page View → segment_view: 62% 손실
   - **원인**: 상단에 세그먼트 탭이 없거나 명확하지 않음
   - **개선**: Hero 섹션에 큰 탭 배치, 스크롤 유도 화살표 추가

2. segment_view → cta_click: 47% 손실
   - **원인**: CTA 버튼이 너무 아래에 있음
   - **개선**: 각 세그먼트 섹션 상단에 CTA 배치

3. cta_click → form_start: 37% 손실
   - **원인**: 폼 페이지 로드 시간 또는 복잡한 폼
   - **개선**: 폼 로드 시간 측정, 필수 필드 최소화

4. form_start → form_submit: 59% 손실 🔴
   - **원인**: 폼 필드 많음, 검증 에러, 모바일 UX 문제
   - **개선**: **긴급** - 폼 이탈률 분석 필요
