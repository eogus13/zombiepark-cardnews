## 목표 설정
- 주제 다양성: 시나리오 강의, 따라하기, 리뷰, Q&A 균형
- 클릭 가능성: 제목에 구체적 숫자나 "왜"를 포함
- 난이도 분산: 쉬운 콘텐츠와 깊이 있는 콘텐츠 교차
- 커뮤니티 반응: 댓글 장려 요소 포함

## 💡 클릭율 높이기 전략

### 높음 (⭐⭐⭐)인 이유
- 구체적 숫자 ("1시간", "3분", "72시간")
- 호기심 갭 ("우리는 뭘 할까?")
- 실용성 ("이것만 기억하세요")
- 약간의 불안감 ("뭐가 다를까?")

### 제목 공식
```
"[상황] + [숫자] + [기대감]"

예:
- 좀비 발생 + 1시간 + 우리가 할 것 = "좀비 발생 첫 1시간"
- 당신의 집 + 7가지 + 안전 체크 = "당신의 집이 안전한가? 7가지 체크리스트"
- 도시 폐쇄 + 72시간 + 식량 = "도시 폐쇄 72시간, 집에서 뭐 먹지?"
```
