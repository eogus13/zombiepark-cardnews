## 1. 채널명 후보 (3개)

### 후보 1️⃣: "생존 수학"
- **강점**: 깔끔하고 기억하기 쉬움. "생존을 과학적으로 배운다"는 뉘앙스
- **약점**: 너무 진지할 수 있음

### 후보 2️⃣: "영흥도 생존 매뉴얼" ⭐ **추천**
- **강점**: 좀비파크(영흥도)를 직접 연결. 구체적이고 고유함. "저 섬은 뭐지?"라는 호기심 유발
- **약점**: 길이가 긴 편. URL이 길어질 수 있음

### 후보 3️⃣: "좀비파크 클래스"
- **강점**: 브랜드명 직접 사용. "클래스"는 강의 느낌으로 신뢰감 있음
- **약점**: 시작 단계에서 브랜드 인지도가 없으면 검색 어려움

**최종 선택**: **"영흥도 생존 매뉴얼"** 또는 **"생존 수학"** (파운더 선호도 확인 필요)

## 2. 채널 소개글 (약 200자)

### 버전 A: "진지하지만 재미있게"
```
좀비 오타쿠가 진지하게 알려주는 생존 강의.
"혹시 모르니까 배워두면 어때?"라는 마음으로
집에서, 도시에서, 위기 상황에서 우리가 할 수 있는
진짜 생존법을 배워봅니다.

실제로 따라 할 수 있고, 가족과 함께 배울 수 있는
생존 매뉴얼을 함께 만들어요.
```

### 버전 B: "그렇다면 해볼까?" (호기심 중심)
```
뉴스에서 나오는 극한 상황들...
만약에 내 주변에서 일어난다면?

진짜로 살아남기 위해 꼭 알아야 할 것들을
쉽고 재미있게 배우는 채널입니다.
응급처치, 식량 확보, 재난 대응까지—
"혹시 모르니까 알아두자"는 생각으로 시작했습니다.
```

## 3. 썸네일 스타일 (템플릿)

### 기본 구성
```
┌─────────────────────────┐
│                         │
│   [큰 텍스트/감정표현]  │
│   (예: "1시간 안에")    │
│                         │
│   [영상 주제 이미지]    │
│   (인물/상황 스샷)      │
│                         │
│   [우측 하단: 로고]     │
│   "영흥도 생존매뉴얼"   │
└─────────────────────────┘
```

### 색상 팔레트
- **주색**: 어두운 빨강 (위험함, 생존감) + 검정 (진지함)
- **강조색**: 노랑 또는 주황 (경고, 호기심)
- **배경**: 검정 또는 진회색 (가독성)

### 요소 규칙
1. **감정 표현**: 당황함, 진지함, 놀람 등을 얼굴 표정으로 표현
2. **숫자 강조**: "72시간", "3분만에" 같은 구체적 숫자는 크게 표시
3. **화살표/강조**: 노란 화살표나 동그라미로 중요 부분 표시
4. **폰트**: 산세리프 (Noto Sans), 굵음, 흰색 테두리로 가독성 확보

### 금지 사항 ❌
- 과도한 이모지 사용
- 너무 밝은 색상 (진지함 훼손)
- 배경 그라데이션만으로 채우기

## 4. 영상 구조 (인트로-본문-아웃트로)

### 📺 인트로 (0~30초)

**목표**: 클릭해서 봐야겠다고 생각하기

```
[화면: 당황스러운 표정 클로즈업]

나레이션 (빠르고 긴장감 있게):
"여러분, 만약에 좀비가 갑자기 나타난다면?
아니면 도시가 폐쇄된다면?
1시간 안에 우리가 할 수 있는 게 뭘까요?"

[화면: 빠르게 영상 메인 컷 3-4개 교차]
[배경음: 긴장감 있는 사운드 이펙트]

"오늘은 그 방법들을 직접 알려드리겠습니다."

[화면: 채널명 로고 2초]
```

### 📚 본문 (30초~9분)

**구조**:
1. **왜 배워야 할까** (1분): 상황 설명, 공감대 형성
2. **핵심 내용** (6분): 단계별 설명, 시각 자료, 직접 시연
3. **주의사항** (30초): "이건 꼭 피하세요"

**톤 유지**:
- 마치 친한 선배처럼 "그런데 말이야..." 스타일
- 너무 전문적이거나 딱딱하지 않게
- 가끔 자조적 유머 ("나도 처음엔 몰랐어요")

### 🎬 아웃트로 (9분~9분 30초)

```
[화면: 따뜻한 톤의 마무리 영상]

나레이션 (부드럽게):
"이 영상이 도움 됐다면, 구독과 좋아요 부탁드려요.
다음 주 [요일]에는 [다음 주제]를 알려드릴 거예요."

[화면: 다음 영상 미리보기 3초]

"그럼 또 봐요!"

[화면: 채널명 로고, 구독 버튼 클로즈업]
[배경음: 밝은 엔딩 사운드]
```

## 5. 톤앤매너 (Tone & Manner)

### 핵심 가치
**"좀비 오타쿠의 진지한 강의"**

### 말투 (Speaking Style)
- ✅ 친근하면서 신뢰감 있음
- ✅ 구체적이고 실용적임
- ✅ 약간의 자조적 유머 포함
- ❌ 학파적이거나 딱딱한 느낌
- ❌ 과장스럽거나 센세이셔널
- ❌ 무섭거나 어두운 톤

### 표현 예시

**"좋은 톤"** ✅
- "여러분, 지금까지 저는 좀비가 올 거라고 생각해요. 안 올 것 같긴 한데..."
- "이건 제가 직접 실험해봤는데, 정말 작동해요."
- "그렇다면 우리가 할 수 있는 게 뭘까?"
- "되게 간단해요. 이것만 기억하세요."

**"피해야 할 톤"** ❌
- "만약 좀비 사태가 난다면 당신은 죽을 겁니다." (너무 공포)
- "이것은 과학적으로 증명된 방법입니다." (너무 딱딱함)
- "하하하, 좀비는 없지만!" (너무 가벼움)

### 캐릭터 설정
- **이름**: "생존매뉴얼 (또는 파운더 실명 사용)"
- **설정**: 평범한 직장인 또는 학생이지만, 혹시 모를 상황에 대비하는 좀비 오타쿠
- **특기**: 생존법을 배우고 실험하는 것이 취미
- **목표**: "우리 모두가 한 번쯤은 배워봐야 할 생존법을 전파하기"

## 6. 채널 운영 규칙

### 업로드 일정
- **초기 (1~6주)**: 주 2-3회 (월/수/금)
- **안정화 (7주 이후)**: 주 1-2회 (일관성 중요)

### 질 관리
- 영상 길이: 8-12분 (최적 시청시간)
- 자막: 한국어 100% (청각장애인, 소음 환경 배려)
- 댓글 응답: 업로드 후 24시간 내 상위 댓글 답변

### 커뮤니티탭 (구독자 500명 이상)
- 주 2-3회 "생존팁" 또는 "질문" 포스트
- 댓글 장려 (알고리즘 상승)
