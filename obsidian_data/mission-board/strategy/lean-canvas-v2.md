# 린 캔버스: 좀비파크 (v2 — 유튜브 퍼스트 전략)

**이전 전략**: 홈페이지 만들고 → 사람 모음
**새로운 전략**: 유튜브로 먼저 사람 모음 → 반응 보고 → 홈페이지 & 오프라인 체험

## 전략 요점

**왜 유튜브부터인가?**
- 돈이 거의 안 든다 (초기 투자 500만원)
- 실패하면 손실이 적다
- 성공하면 "사람들이 먼저 묻는" 상황을 만든다
- 콘텐츠가 자산이 된다 (재사용, 재수익화 가능)

**홈페이지는 언제 만드나?**
- 댓글/문의에서 "어디서 할 수 있어요?"가 월 10건 이상 나올 때
- = 수요가 검증된 시점
- 그때 홈페이지 + 소규모 오프라인 체험 시작

**실패 시나리오 대응**
- 만약 구독자가 5,000명에서 멈추면? → 콘텐츠 방향 조정 또는 중단
- 만약 댓글에서 수요가 없으면? → 다른 접근법 테스트
- 시간 손실: 3~6개월 / 금전 손실: 500만원 (충분히 허용 범위)
