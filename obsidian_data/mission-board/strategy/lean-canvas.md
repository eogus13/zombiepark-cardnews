# 린 캔버스: 좀비파크

| 영역 | 내용 |
|------|------|
| **문제** | 1. "살 수 있는" 실전 생존 기술을 배울 수 없음 (일반 서바이벌 콘텐츠는 놀이 수준)<br>2. 극한 상황 대비에 대한 실제 자신감과 경험이 부족함<br>3. 가족 단위의 몰입감 있는 체험 콘텐츠 부족 |
| **고객 세그먼트** | 얼리어답터:<br>• 20-40대 생존 관심층 (MZ세대 밀리터리 매니아, 프레퍼)<br>• 가족 단위 체험 관광객 (교육적 체험에 관심)<br>• 기업 팀빌딩/리더십 프로그램 수요층 |
| **고유 가치 제안** | "이걸 할 줄 알면 산다" — 디즈니랜드의 몰입감으로 배우는 실전 생존 기술 |
| **솔루션** | 1. 단계별 자격증 시스템 (1~5등급): 재방문 유도 + 접근 영역 확대<br>2. VIP 서바이벌 프로그램: 72시간 숙박형 극한 훈련<br>3. 4대 라이프라인 어트랙션 (물/식량/에너지/의료): 실제로 작동하는 생존 인프라<br>4. 영흥도 섬 특성 활용: 완전 격리 시뮬레이션 |
| **채널** | 1. SNS/유튜브: 밀리터리/생존 커뮤니티 타게팅 (B롤, 리뷰)<br>2. 기업 웰니스/팀빌딩 에이전시 직접 영업<br>3. 관광 매체: 인천 이색 체험 관광지로 포지셔닝<br>4. 프리퍼 커뮤니티 (온라인 포럼, 팟캐스트) |
| **수익 모델** | 1. 입장료 (당일 1등급): 50,000~70,000원<br>2. 등급 상승 미션비: 급당 80,000~150,000원<br>3. VIP 카드 (연간): 2,000,000~5,000,000원<br>4. 기업 팀빌딩 패키지: 5,000,000~20,000,000원 (단체)<br>5. IP 라이선싱 (생존 매뉴얼 자격증): 추후 B2B 확대 |
| **비용 구조** | 1. 초기 인프라: 영흥도 부지 조성, 시설 건설 (수십억 규모)<br>2. 운영비: 직원(가이드, 안전팀), 장비 유지, 물품<br>3. 마케팅: SNS, 인플루언서, 기업 영업<br>4. 보험/안전: 생존 훈련 특화 보험 |
| **핵심 지표 | KPI)** | 1. 월 방문객 수: 초기 500명 → 12개월 2,000명<br>2. 재방문율: 30% (등급 상승)<br>3. VIP 전환율: 방문객의 5~10%<br>4. 평균 고객 생애가치(LTV): 1,000,000원 (여러 번 방문)<br>5. 등급 수료 완료율: 60% (이탈 방지 지표) |
| **압도적 우위** | 1. 영흥도 지리적 고립성 (영흥대교 1개 접근로만 → 완벽한 시뮬레이션)<br>2. 현지 인프라 보유 (발전소, 갯벌, 농경지, 곤충 사육 노하우)<br>3. 엔터테인먼트 + 실전성 결합 (경쟁사는 둘 중 하나만)<br>4. IP 자격증 시스템 (반복 방문 + 장기 고객 확보) |
