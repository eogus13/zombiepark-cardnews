# 단계별 로드맵: 좀비파크 (v2 — 유튜브 → 오프라인 → 테마파크)

---

## 전체 타임라인

```
Phase 1 (0~3개월)    → Phase 2 (3~6개월)    → Phase 3 (6~12개월)   → Phase 4 (12개월~)
유튜브 채널          커뮤니티 + 소체험      정식 프로그램            테마파크 투자 유치
```

---

## Phase 1: 유튜브 채널 (첫 3개월)
**목표**: 구독자 5,000명 + 월 인지도 구축

### 1-1. 준비 (주 1~2)
- [ ] 채널 개설 + 프로필 세팅 (페르소나: "좀비 오타쿠")
- [ ] 카메라/음성 장비 구입 (200만원)
- [ ] 촬영/편집 노하우 습득 (유튜브 튜토리얼)
- [ ] 영상 콘셉트 10개 기획
- [ ] 스튜디오 세팅 (집에서 또는 렌탈)

### 1-2. 첫 콘텐츠 (주 3~12)
**콘셉트**: "만약에 이런 일이 생기면?"
- 영상 1: "좀비 발생 첫 1시간 생존법"
- 영상 2: "도시 폐쇄 시 식량 확보 (집에서 7일 버티기)"
- 영상 3: "응급처치 강의 (가족이 꼭 알아야 할 것)"
- 영상 4: "생존용품 리뷰 — 이건 사지 마라"
- 영상 5~10: 더 만들기...

**배포 규칙**
- 매주 월/수/금 3개씩 (또는 월/금 2개씩)
- 각 영상 8~12분 (유튜브 최적 길이)
- 썸네일/제목: 클릭 유도형 ("절대 하면 안 되는 이유", "의외의 진실")

### 1-3. 성과 지표
| 지표 | 초기 | 목표 (3개월) |
|------|------|-----------|
| 구독자 | 0 | 5,000명 |
| 월 조회수 | 0 | 5만~10만 |
| 평균 시청 완료율 | - | 50% 이상 |
| 댓글 "어디서 해봐요?" | 0 | 월 2~5건 |

### 1-4. 주의사항
- 초기 3개월은 "마음대로" 업로드 (구독자 1,000 이상까지 광고 못 함)
- 장기 게임이라 생각하기
- 퀄리티 > 빈도 (월 4개 좋은 영상이 월 20개 나쁜 영상보다 낫다)

## Phase 2: 커뮤니티 + 소규모 오프라인 체험 (3~6개월)

**진입 조건**: 구독자 5,000명 + 댓글 월 5건 이상 "어디서 해봐요?"

### 2-1. 온라인 확장 (동시진행)
- [ ] 유튜브 커뮤니티탭 활성화
  - 주 3~4회 "생존팁" 포스트
  - 시청자 질문에 직접 답변
- [ ] 온라인 굿즈 출시 (선택)
  - "좀비파크 생존 매뉴얼" (PDF) 판매
  - 가격: 5,000~10,000원
- [ ] 유료 멤버십 (유튜브 또는 별도 커뮤니티)
  - 월 2,990원 (영상 공개 전 미리 보기 등)

### 2-2. 오프라인 소체험 (시험 운영)
**컨셉**: "유튜브에서 배운 것을 실제로 해보자"

- 위치: 서울 근처 협력 시설 대여 (예: 캠핑장, 체험 센터)
- 규모: 월 2회, 1회당 10~20명
- 프로그램: 3시간 (예: 물 정수하기 + 응급처치 실습)
- 가격: 5만~10만원
- 목적:
  - 온라인 팬을 오프라인으로 전환 경험
  - 피드백 수집 ("어떤 내용을 더 배우고 싶어?")
  - 향후 영흥도 프로그램의 프로토타입

### 2-3. 성과 지표
| 지표 | 목표 (6개월) |
|------|-----------|
| 구독자 | 10,000명 |
| 월 조회수 | 20만~50만 |
| 월 문의 "어디서 해봐요?" | 10건 이상 |
| 소체험 신청 | 월 10명 이상 |
| 커뮤니티 월간 활성 사용자 | 500명 |
| 월 수익 (광고+굿즈+체험) | 200~300만원 |

### 2-4. 주의사항
- 아직 영흥도 시설 건설 중 (완성까지 기다릴 필요 없음)
- 소체험은 "검증 목적" — 수익 목표가 아님
- 다음 단계 근거 수집

## Phase 3: 정식 프로그램 + 홈페이지 (6~12개월)

**진입 조건**: 구독자 20,000명 이상 + 월 문의 20건 이상

### 3-1. 홈페이지 & 예약 시스템
- [ ] 간단한 홈페이지 (Webflow 또는 Wix)
  - 소개 페이지
  - 유튜브 채널 링크
  - 체험 예약 폼
- [ ] 예약 관리 시스템 (Calendly, Notion)

### 3-2. 본격 오프라인 체험 프로그램 (영흥도)

**만약 영흥도 시설이 완공되었다면:**
- 당일 체험 프로그램 (1등급)
  - 가격: 5만~7만원
  - 시간: 4시간 (또는 전일)
  - 참석: 주말 월 2회 × 30명 = 월 60명

**아직 시설이 미완공이라면:**
- 협력 시설에서 계속 소규모 체험 운영
- 영흥도 완공 시 오픈 예정 공지
- 사전 회원 등록 시작 ("오픈 대기자 명단")

### 3-3. 유튜브 콘텐츠 고도화
- 정식 체험 프로그램 영상 (수강자 후기)
- "체험자 인터뷰" (그들의 변화 이야기)
- 정기 라이브 (커뮤니티와 실시간 Q&A)

### 3-4. 성과 지표
| 지표 | 목표 (12개월) |
|------|-----------|
| 구독자 | 50,000명 |
| 월 조회수 | 100만 이상 |
| 월 체험 신청 | 100명 |
| 월 수익 | 500만~1,000만원 |
| 커뮤니티 활성 멤버 | 2,000명 |

## Phase 4: 테마파크 투자 유치 (12개월~)

**진입 조건**:
- 구독자 50,000명 이상
- 월 체험 신청 100명 이상 (수요 증명)
- 온라인 커뮤니티 2,000명 이상 (팬덤 증명)

### 4-1. 투자 유치 신청
- 대상: 테마파크 개발사, 관광 펀드, 건설사 등
- 근거 자료:
  - 유튜브 채널 데이터 (구독자, 조회수)
  - 고객 만족도 리뷰 (체험자 평가)
  - 커뮤니티 반응 데이터
  - 초기 매출 데이터 (월 500만~1,000만원)

### 4-2. 영흥도 본격 건설 + 확장
- 단계별 구역 오픈 (안전을 위해 한두 구역부터)
- 등급 시스템 도입 (1~5등급)
- VIP 숙박 프로그램 (72시간 생존 시뮬레이션)

### 4-3. 다각화
- 기업 팀빌딩 패키지 (B2B 영업 본격화)
- 생존 자격증 교재/온라인 강의 (IP 활용)
- 굿즈 & 인테리어 브랜드 확장
