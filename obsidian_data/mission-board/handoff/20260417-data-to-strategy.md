### 초기 4주 (5월 1주차 ~ 4주차)

**데이터 기반 의사결정 포인트:**

#### 포인트 1: 세그먼트별 성과 불균형

```
시나리오:
- 프레퍼층: 20% 트래픽, 8% 전환율 → 강세
- 가족층: 40% 트래픽, 3% 전환율 → 약세
- 기업층: 15% 트래픽, 2% 전환율 → 매우 약세

데이터 분석가의 피드백:
"가족층의 이탈율이 높고(45%), 기업층은 섹션 도달도 낮습니다(19%).
가족 섹션의 카피/UX를 재검토하고, 기업 섹션을 상단으로 이동할 것을 권장합니다."

전략가의 의사결정:
1) 초기 전략 유지? → "가족층은 아직 리드 제너레이션 단계. 추가 예산 투자하기"
2) 전략 수정? → "프레퍼층에 집중하고, 가족층은 2차 캠페인으로 미루기"
3) 고객 가정 재검토? → "실제 가족층 수요가 예상보다 낮은가?"

→ 린 캔버스 "고객 세그먼트" 우선순위 조정 필요

결정: P(가정) = 40% → 25%로 조정
       P(프레퍼) = 30% → 50%로 강화
```

#### 포인트 2: 채널별 성과 편차

```
시나리오:
- Organic Search: 30% 트래픽, 6.2% 전환율 ✅ 최고
- Instagram: 35% 트래픽, 6% 전환율 ✅ 양호
- Facebook: 15% 트래픽, 3% 전환율 🔴 낮음
- Email: 10% 트래픽, 4% 전환율 🟡 중간

데이터 분석가의 피드백:
"Organic Search가 가장 효율적입니다. 'SEO 친화적 콘텐츠'를 추가로 만들면,
광고비 없이도 트래픽을 늘릴 수 있습니다."

전략가의 의사결정:
1) 채널 전략 수정
   - "SNS/유튜브 중심" → "유기검색 + SNS 균형"
   - Google Ads 예산 추가 배분

2) 콘텐츠 전략 수정
   - "생존 기술 블로그" 추가 작성
   - "생존 체험 가이드" 콘텐츠 시리즈화

→ 린 캔버스 "채널" 조정: 유기검색 35%, SNS 40%, Email 15%, PR 10%
```

#### 포인트 3: 가격 전략 재검토

```
시나리오:
- 현재 예약가: 50,000~70,000원
- CPA (고객획득비): 25,000원
- LTV (고객생애가치): 계산 필요 (재방문 데이터 필요)

데이터 분석가의 피드백:
"현재 CPA 25,000원, 1차 예약 평균값 50,000원입니다.
LTV를 계산하려면 재방문율(목표 30%)을 추적해야 합니다."

전략가의 의사결정:
1) LTV 계산
   - LTV = 첫 구매 + (재방문 수 × 평균값)
   - 50,000 + (30% × 3회 방문 × 60,000) ≈ 104,000원

2) CAC 대비 LTV 분석
   - CAC: 25,000원, LTV: 104,000원
   - LTV:CAC ratio = 4.16 (목표 3.0 이상) ✓ 양호

3) 가격 최적화 기회
   - 현재 입장료만 50,000원
   - 상향 여지가 있는가? (데이터 검증 필요)
   - A/B 테스트: 50,000 vs 60,000 비교

→ 가격 정책 결정: 50,000원 유지 또는 상향 A/B 테스트
```

## 첫 달의 예상 시나리오

### Scenario A: 데이터가 가정을 지지하는 경우 ✅

```
실제: 프레퍼층 50% 도달, 8% 전환율
예상: 프레퍼층 30-40% 도달, 5-7% 전환율

전략가 액션:
- "우리 가정이 정확하다. 프레퍼층에 더 투자하자"
- 프레퍼층 광고비 +50%
- 프레퍼층 콘텐츠 확대
- 다른 채널도 프레퍼층 타게팅 추가
```

### Scenario B: 일부 가정이 틀린 경우 🟡

```

전략가 액션:
- "기업층 수요가 우리 예상보다 낮다"
- 기업층 온라인 광고 축소
- 직접 영업(이메일, 파트너)으로 전환
- 린 캔버스에서 기업층 비중 조정 (30% → 15%)
```

### Scenario C: 새로운 기회 발견 🚀
