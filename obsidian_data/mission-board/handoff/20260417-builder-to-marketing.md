```

    가이드: "생존 기술, 진짜로 배울 곳"
    또는?

[ ] 문제 공감 문장 (2~3줄)
    예시:
    "유튜브 강좌는 검증이 안 된다."
    "비싼 해외 프로그램은 현실적이지 않다."
    "제대로 배우고 싶지만 한계가 있다."

[ ] 가치 포인트 3가지 (각 1줄 헤드라인 + 부연 2줄)

       → 설명: 재방문 유도, 실력 증명, 커뮤니티 지위
    2. "실제로 작동하는 생존 인프라"
       → 설명: 자전거 발전기, 정수 시스템 등 실전성
    3. "프레퍼 커뮤니티와의 네트워킹"
       → 설명: 같은 관심층과 경험 공유, 팁 교환

[ ] CTA 버튼 텍스트
    현재: "프레퍼 스페셜 패키지 보기"
    또는?
```
