## 핸드오프 개요

마케팅에서 유튜브 채널 전략을 완성했습니다. 이제 데이터 분석가가 **"우리가 성공하는지를 어떻게 측정할 것인가"**를 설계해야 합니다.

### 마케팅이 만든 산출물
1. ✅ 유튜브 채널 기획서 (채널명, 톤앤매너, 썸네일, 영상 구조)
2. ✅ 콘텐츠 리스트 (첫 달 20개 영상 기획)
3. ✅ 첫 영상 대본 (실제 촬영 가능한 수준)

### 이제 데이터 분석가가 설계할 것
- 📊 **핵심 지표 (KPI)**: 성공/실패를 판단하는 구체적 숫자
- 📈 **트래킹 방법**: 각 지표를 어떻게 측정할 것인가
- ⚙️ **자동화**: 대시보드 또는 리포트 자동화
- 🎯 **의사결정 기준**: "어느 숫자부터 다음 단계로 가는가"

## Phase 1 (0~3개월) 성공 기준

### 🎬 YouTube 채널 수준의 KPI

| KPI | 초기 | 월 1 | 월 2 | 월 3 (목표) | 측정 방법 |
|-----|------|------|------|-----------|---------|
| **구독자** | 0 | 50~200 | 200~500 | 3,000~5,000 | YouTube Analytics → 일일 추적 |
| **월 총 조회수** | 0 | 500~2,000 | 2,000~10,000 | 20,000~50,000 | YouTube Analytics → 월간 합계 |
| **평균 시청시간** | - | 10~30 분 | 30~60 분 | 100+ 분 | YouTube Analytics → 월간 |
| **평균 완료율** | - | 30% | 40% | 50% 이상 | YouTube Analytics → 영상별 |
| **클릭률 (CTR)** | - | 2~3% | 3~5% | 5% 이상 | YouTube Analytics → 영상별 |
| **댓글 수** | - | 영상당 2~5개 | 영상당 5~10개 | 영상당 10+ 개 | YouTube 직접 계산 또는 API |

## 📊 세부 측정 기준 (Detailed Metrics)

### 1️⃣ **채널 성장 (Channel Growth)**

**목표**: 신뢰도 있는 초기 커뮤니티 형성

```
구독자 성장률 (Weekly):
- 주 평균: +10~50명
- 가입: 영상 개수와 조회수 대비 선형 성장
- 이탈: 월 5% 이하 (정상)

신규 구독자 유입 경로 (분석할 것):
- YouTube 알고리즘 추천 (Browse Features): 40% 이상
- 검색 (Search): 20% 이상
- 외부 링크: 10% (유튜브 커뮤니티, SNS 등)
- 직접 (Direct): 10% (채널 재방문)
```

### 2️⃣ **콘텐츠 성과 (Content Performance)**

**목표**: 어느 주제가 인기 있는지, 어느 형식이 효과적인지 파악

```
영상별 성과표:
- 영상번호, 제목, 업로드일
- 조회수 (매일 측정)
- 댓글 수 (유형별 분류)
- 좋아요 수
- 공유 수
- 클릭률 (CTR)
- 평균 시청시간
- 완료율

분석 기준:
1. "어떤 제목이 클릭을 더 받나?" (CTR 비교)
   예: "#1 (1시간) vs #4 (72시간)" → 숫자가 있는 제목이 높은 CTR

2. "어떤 주제를 시청자가 끝까지 봤나?" (완료율)
   예: "따라하기" 영상이 "설명" 영상보다 완료율 높음

3. "어떤 영상이 댓글을 유도했나?" (댓글 수)
   예: "Q&A" 형식이 실시간 질문 댓글 증가
```

### 3️⃣ **시청자 참여 (Audience Engagement)**

**목표**: 일회성 시청자가 아닌 "팬"을 만드는가?

```
댓글 분석 (주간):
- 댓글 총 개수
- 긍정 vs 부정 비율 (센티먼트 분석)
- 질문 댓글 비율
- "어디서 해봐요?" (구매의도 신호) 댓글 개수
  → 이게 월 5개 이상이면 Phase 2 진입 신호!

좋아요 비율 (Like Rate):
- 좋아요 수 / 조회수 = 좋아요율
- 목표: 1~2% (한국 YouTube 평균)
- 높을수록 콘텐츠 만족도 높음

구독률 (Subscribe Rate):
- (신규 구독자 / 조회수) × 100
- 목표: 1% 이상
- 영상이 좋을수록 높아짐
```

### 4️⃣ **수익 신호 (Monetization Signals)**

**목표**: 이 채널이 수익화될 가능성이 있는가?

```
광고수익 (구독자 1,000명 이상):
- 일일 수익 추적 (RPM: 매출/천 조회수)
- 한국 YouTube 평균 RPM: 2,000~5,000원
- 우리 목표: 월 30만원 (= 월 조회수 50만 × 평균 RPM)

공유 확산률 (Share Rate):
- 공유 수 / 조회수
- 높을수록 "남에게 추천할 만한 콘텐츠"

외부 트래픽:
- YouTube에서 우리 채널로 오는 사람 비율
- 향후: 블로그, SNS로 유입되는 클릭 추적
```

## 🎯 의사결정 기준 (Decision Checkpoint)

### Phase 1 → Phase 2 진입 조건

**다음 중 1가지 이상이 달성되면 오프라인 체험 시작 검토:**

```
✅ 구독자 5,000명 도달
✅ 월 조회수 50,000 이상 유지 (2주 연속)
✅ 댓글에서 "어디서 해봐요?" 월 5건 이상
✅ 평균 완료율 50% 이상 (6개 이상 영상)
✅ 좋아요율 평균 1% 이상

❌ 실패 신호:
- 구독자 500명에서 3주 이상 정체
- 월 조회수 2,000 이하 유지
- 평균 완료율 30% 이하
- 댓글에 부정적 의견 30% 이상
```

## 📈 트래킹 대시보드 설계 요청

### 필요한 데이터 수집

**자동 수집 (YouTube API):**
```
- 일일 기준:
  * 채널 구독자 수
  * 총 조회수
  * 수익 (RPM)

- 영상별 기준:
  * 조회수 (누적)
  * 시청시간 (누적)
  * 완료율
  * CTR
  * 댓글 수
  * 좋아요 수
  * 공유 수
  * 상위 댓글 텍스트 (감정 분석용)
```

**수동 수집/분석:**
```
- 주간 (분석가가 직접):
  * 댓글 감정 분석 (긍정/부정/중립)
  * "어디서 해봐요?" 댓글 빼내기
  * 인플루언서 언급 여부
  * 경쟁 채널 비교 (유사 채널 조회수 비교)
```

### 대시보드 형태

**Option 1: Google Sheets (추천)**
- YouTube Analytics 데이터를 자동 수집 (API 연결)
- 매일 갱신, 지난 30일 그래프 표시
- 임계값 도달 시 색상 변경 (자동 경고)

**Option 2: Data Studio**
- YouTube 계정 연결 → 자동 시각화
- 모든 팀원이 실시간 확인 가능

**Option 3: Notion 통합**
- 간단한 트래킹용
- 주간 리포트 자동 생성

## ⚠️ 주의사항

### 1. 초기 3개월은 "절대 수익"이 아니라 "성장률"로 봐야 함

```
❌ "이번 달 수익이 0원이면 실패다"
✅ "이번 달 조회수가 지난달의 3배면 성공이다"

초기 YouTube는 구독자 1,000명 이상이어야 광고 활성화
우리는 월 3개월 동안 월 0원 수익 가능성 있음
그래도 괜찮음 (성장하면 나중에 수익 나옴)
```

### 2. "조회수" 하나만으로는 판단하지 말 것

```
❌ "조회수 500회밖에 없네, 실패했다"
✅ "조회수 500회지만 완료율 70%, 댓글 20개 = 매우 우호적"

중요한 건:
- 조회수 × 완료율 = "실제로 끝까지 본 시청자"
- 댓글 수 / 조회수 = "팬 생성율"
- "어디서 해봐요?" 댓글 = "구매의도"
```

### 3. 초기 불규칙성은 정상

```
주 1 영상 vs 주 3 영상: 조회수가 다름
새로운 채널이라 알고리즘이 불규칙함
"안정화"는 3개월 후부터

그래서 지표를 보고 판단할 때:
- 월간 기준으로 본다 (주간 변동성 무시)
- 최소 3개 영상 이상의 평균을 본다
```
