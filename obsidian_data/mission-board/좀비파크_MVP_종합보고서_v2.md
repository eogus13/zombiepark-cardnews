# 좀비파크 MVP 종합보고서 (v2 — 유튜브 퍼스트)

**작성일**: 2026-04-17

---

## 한줄 요약

홈페이지 먼저 만드는 대신, **유튜브에서 "좀비 오타쿠" 생존 강의**를 먼저 시작해서 사람들이 진짜 관심 있는지 확인하자.

---

## 1. 왜 전략을 바꿨는지

### 이전 계획 (v1)
홈페이지 만들고 → 예약 받고 → 반응 보자

### 문제점
사람들이 예약하면? **우리가 보여줄 게 아직 없다.** 테마파크는 아직 짓지도 않았으니까.

### 새 계획 (v2)
**유튜브로 먼저 시작한다.**
- "좀비 오타쿠"로 생존 강의 영상을 올린다
- 사람들이 모이면 → 소규모 오프라인 체험을 해본다
- 반응이 좋으면 → 그때 홈페이지 만들고, 영흥도 테마파크를 진짜 시작한다

### 왜 이게 더 좋은지
- **돈이 거의 안 든다** (카메라 + 편집만 있으면 됨)
- **실패해도 손해가 적다** (최대 500만원)
- **성공하면 사람들이 먼저 "어디서 해봐요?" 묻는다** → 그때 만들면 된다
- **영상 자체가 자산이 된다** (나중에 교재, 강의, 기업교육으로 재활용)

## 2. 뭘 하는 건지 (4단계)

```
1단계 (0~3개월)    →  2단계 (3~6개월)    →  3단계 (6~12개월)   →  4단계 (12개월~)
유튜브 채널           커뮤니티 + 소체험      홈페이지 + 정식운영     테마파크 투자유치
```

### 1단계: 유튜브 채널 (처음 3개월)

**한마디로**: "좀비 오타쿠"로 유튜브에서 생존 강의를 시작한다.

**뭘 올리냐면**:
- "만약에 좀비가 나타나면 첫 1시간에 뭘 해야 할까?"
- "도시가 폐쇄됐을 때 집에서 72시간 버티는 법"
- "손으로 정수 필터 만들기 (3분이면 됨)"
- "생존용품 100만원짜리 vs 1만원짜리, 뭐가 다를까?"

이런 영상을 **매주 2~3개** 올린다.

**3개월 후 목표**:
- 구독자 5,000명
- 월 조회수 5만~10만
- 댓글에 "어디서 해볼 수 있어요?" 2~5건

### 2단계: 커뮤니티 + 소규모 체험 (3~6개월)

**시작 조건**: 구독자 5,000명 넘고, "어디서 해봐요?" 댓글이 계속 나올 때

**뭘 하냐면**:
- 유튜브 커뮤니티에서 팬들과 소통
- 서울 근처 캠핑장 같은 데서 **월 2회, 10~20명 소규모 체험** 운영
- 3시간짜리 (물 정수 + 응급처치 실습), 가격 5~10만원
- 이건 돈 벌려는 게 아니라 **"사람들이 진짜 오프라인도 원하는지"** 확인하는 거

**6개월 후 목표**:
- 구독자 10,000명
- 소체험 신청 월 10명 이상
- 월 수익 200~300만원 (광고 + 체험비)

### 3단계: 홈페이지 + 정식 프로그램 (6~12개월)

**시작 조건**: 구독자 20,000명 넘고, 월 문의 20건 이상

**뭘 하냐면**:
- 이제 홈페이지를 만든다 (체험 예약용)
- 영흥도에서 정식 당일 체험 프로그램 운영 (주말 월 2회, 30명)
- 유튜브에 체험 후기 영상도 올린다

**12개월 후 목표**:
- 구독자 50,000명
- 월 체험 신청 100명
- 월 수익 500~1,000만원

### 4단계: 테마파크 투자 유치 (12개월 이후)

**시작 조건**: 구독자 50,000명 + 월 체험 100명 + 커뮤니티 2,000명

이 숫자가 나오면 → **투자자한테 보여줄 수 있는 근거**가 생긴다.
"이만큼 사람이 관심 있고, 이만큼 돈을 냅니다" → 영흥도 본격 건설 시작

## 3. 유튜브 채널 어떻게 만드는지

### 채널 이름 후보
1. **"영흥도 생존 매뉴얼"** ← 추천 (영흥도를 자연스럽게 알림, 호기심 유발)
2. "생존 수학" (깔끔하지만 좀 딱딱)
3. "좀비파크 클래스" (브랜드 직접 쓰지만 아직 인지도 없음)

### 채널 소개글
```
좀비 오타쿠가 진지하게 알려주는 생존 강의.
"혹시 모르니까 배워두면 어때?"라는 마음으로
집에서, 도시에서, 위기 상황에서 우리가 할 수 있는
진짜 생존법을 배워봅니다.
```

### 말투 (톤)
- **좋은 예**: "여러분, 저는요... 좀비가 올 거라고 생각해요. 안 올 것 같긴 한데..."
- **안 좋은 예**: "만약 좀비 사태가 난다면 당신은 죽을 겁니다" (너무 무섭)
- **핵심**: 친한 선배가 말해주는 느낌. 진지하지만 가끔 자조적 유머.

### 영상 구조 (매 영상 이렇게)
1. **인트로 (30초)**: 호기심 당기는 질문 던지기
2. **본문 (8분)**: 단계별로 쉽게 설명 + 직접 보여주기
3. **아웃트로 (30초)**: 구독 부탁 + 다음 영상 예고

### 썸네일 스타일
- 배경: 어두운 빨강 + 검정 (생존 느낌)
- 강조: 노란색으로 숫자 크게 ("72시간", "3분")
- 얼굴: 당황/진지한 표정 클로즈업

## 4. 첫 달에 올릴 영상 20개

전체 리스트에서 **특히 중요한 것 5개**:

| 순서 | 제목 | 왜 중요한지 |
|------|------|-----------|
| 1번 | "좀비 발생 첫 1시간, 우리는 뭘 해야 할까?" | 채널 오프닝. 시청자를 확 끌어야 함 |
| 3번 | "정수 필터를 손으로 만들기 (3분 안에)" | 따라하기 콘텐츠. 참여도 높음 |
| 4번 | "도시 폐쇄 72시간, 집에서 뭐 먹지?" | 누구나 공감. 실용적 |
| 6번 | "응급처치를 정말 몰라도 되나요?" | 가족 시청자 끌어옴 |
| 12번 | "가족과 함께할 수 있는 생존 게임 4가지" | 가족 타겟 확보 |

나머지 15개도 준비돼 있어. 생존용품 리뷰, 지역별 재난 분석, 시청자 Q&A 등.

**업로드 일정**: 매주 월/수/금 (주 3개씩, 7주면 20개 완료)

## 5. 첫 영상 대본 (이미 다 써놨음)

**제목**: "좀비 발생 첫 1시간, 우리는 뭘 해야 할까?"
**길이**: 약 11분

**내용 요약**:
- **0~30분**: 정보 확인 (뉴스, 가족 위치, 이동 경로)
- **30~50분**: 안전한 곳 정하기 (집? 회사? 길거리?)
- **50~60분**: 이동 준비 (운동화, 필수품 3개, 탈출 경로)

대본 전문은 `mission-board/marketing/first-video-script.md`에 있어.

## 6. 돈 얼마나 드는지

### 1단계 비용 (유튜브 시작)

| 항목 | 비용 | 꼭 필요? |
|------|------|---------|
| 카메라 + 삼각대 | 100~200만원 | ⭕ (스마트폰으로 시작해도 됨) |
| 마이크 | 10~30만원 | ⭕ |
| 편집 프로그램 | 월 10만원 | ⭕ |
| 조명 | 10~30만원 | △ (자연광으로 가능) |
| 배경/세트 | 50~100만원 | △ (집에서 가능) |
| **합계** | **200~500만원** | |

**스마트폰으로 시작하면?** → 마이크(10만원) + 편집(10만원) = **20만원**으로도 가능

### v1 대비 비교

| | v1 (홈페이지) | v2 (유튜브) |
|---|---|---|
| 초기 비용 | 4,000원 | 20~500만원 |
| 실패 시 손실 | 시간만 날림 | 장비는 남음 |
| 성공 시 자산 | 홈페이지 1개 | 영상 수십 개 (계속 조회됨) |
| 수요 검증 방법 | 예약 버튼 클릭 수 | 구독자 + 댓글 반응 |

## 7. 성공이란?

| 시점 | 이 숫자가 나오면 성공 | 다음 할 일 |
|------|---------------------|-----------|
| 1개월 | 구독자 300명, 영상 15개 | 계속 올린다 |
| 3개월 | 구독자 5,000명, "어디서 해봐요?" 댓글 5건 | 소규모 오프라인 체험 시작 |
| 6개월 | 구독자 10,000명, 체험 신청 월 10명 | 홈페이지 만들기 시작 |
| 12개월 | 구독자 50,000명, 월 수익 500만원 | 투자 유치 준비 |

안 되면? → 영상 방향 바꿔보고, 3~6개월 더 해봐도 **최대 손실 500만원**. 충분히 감당 가능.

## 8. 형이 직접 해야 할 일

### 이번 주 (지금 당장)

**1. 채널 이름 정하기**
- "영흥도 생존 매뉴얼" vs "생존 수학" vs "좀비파크 클래스"
- 직감으로 골라도 됨 (나중에 바꿀 수 있음)

**2. 유튜브 채널 만들기**
- youtube.com에서 "채널 만들기"
- 위에 적어둔 소개글 붙여넣기
- 프로필 사진은 나중에 다듬어도 됨, **일단 이름부터 잡기**

**3. 인스타그램 계정 만들기**
- @zombiepark.kr 또는 채널명에 맞게
- 이름 선점이 중요!

**4. 촬영 장비 결정하기**
- **최소**: 스마트폰 + 핀마이크 (10만원) → 바로 시작 가능
- **기본**: 카메라(100만원) + 마이크(20만원) + 조명(20만원)
- 어떤 걸로 할지 정하기

### 다음 주 (4/22~29)

**5. 첫 영상 촬영하기**
- 대본은 이미 써놨음 (11분짜리)
- 3~4번 소리 내서 읽어보기 (자연스러워질 때까지)
- 촬영 → 편집 → 자막 넣기
- 모르겠으면 "마케터한테 도와달라" 하면 가이드해줌

**6. 두 번째, 세 번째 영상 기획하기**
- 2번: "당신의 집이 안전한가? 7가지 체크리스트"
- 3번: "정수 필터를 손으로 만들기 (3분 안에)"
- 콘텐츠 리스트 20개는 이미 다 정리해놨음

### 론칭 주 (4/29~5/5)

**7. 첫 영상 업로드!**
- 제목: "좀비 발생 첫 1시간, 우리는 뭘 해야 할까?"
- 썸네일: 당황한 표정 + "1시간" 큰 글씨 + 어두운 빨강 배경
- 인스타에도 15초 숏폼 버전 올리기
- 지인들한테 공유 부탁

**8. 매주 확인하기**
- 조회수 얼마나 나왔는지
- 댓글에 뭐라고 쓰는지 (특히 "어디서 해봐요?" 류)
- 시청 완료율 (유튜브 스튜디오에서 볼 수 있음)
- "데이터 분석가한테 리포트 만들어달라" 하면 정리해줌

## 9. 리스크 (잘 안 될 수 있는 것들)

| 걱정되는 것 | 얼마나 가능? | 그러면 어쩌지? |
|------------|-----------|-------------|
| 구독자가 안 늘어남 | 가능성 있음 | 영상 주제/스타일 바꿔보기, 숏폼 늘리기 |
| 오프라인 체험에 사람이 안 옴 | 낮음 (이미 온라인에서 확인하고 시작) | 온라인만 계속 하기 |
| 영흥도 시설 건설이 늦어짐 | 가능성 있음 | 서울 근처 다른 장소에서 계속 운영 |
| 비슷한 채널이 나타남 | 가능성 있음 | "좀비 오타쿠"라는 독특한 캐릭터로 차별화 |
