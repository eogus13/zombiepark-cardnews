# 02. 만약 감염된다면 — 상황별 대응 매뉴얼
> "감염이 어떤 형태로 오든, 살아남는 법은 배울 수 있다."
> 이 문서는 확정된 규칙이 아니라, 다양한 시나리오에 대한 대응 가이드다.

---

## 이 문서를 읽기 전에

아직 좀비 사태는 일어나지 않았다.
하지만 **감염이 온다면** 어떤 형태이든 공통적으로 해야 할 일이 있고,
형태에 따라 다르게 대응해야 할 부분이 있다.

---

## PART 1: 어떤 상황이든 통하는 생존 원칙

### 원칙 1 — 상황 인지가 먼저다
> "무슨 일이 벌어지고 있는지 모르면, 아무것도 할 수 없다."

- 뉴스, 라디오, 주변 상황을 즉시 파악
- 사람들이 패닉인지, 공격적인지, 아픈지 관찰
- 가장 중요한 정보: **무엇이 전파되는가? 어떻게 전파되는가?**

### 원칙 2 — 첫 24시간이 생사를 가른다

| 우선순위 | 해야 할 일 | 이유 |
|---------|-----------|------|
| 1 | 안전한 공간 확보 | 위협으로부터 물리적 차단 |
| 2 | 물 확보 | 사람은 3일 못 마시면 죽는다 |
| 3 | 출입구 봉쇄/바리케이드 | 침입 방지 |
| 4 | 통신 수단 확보 | 정보 = 생존 |
| 5 | 무기/방어 도구 확보 | 최후의 수단 |
| 6 | 동료 확보 | 혼자보다 함께가 생존 확률 높음 |

### 원칙 3 — 소리를 내지 마라
어떤 위기 상황에서든 소리는 위치를 알린다.
패닉 군중, 약탈자, 감염자 — 모두 소리가 나는 곳으로 몰린다.

### 원칙 4 — 높은 곳을 잡아라
대부분의 위협은 아래에서 온다. 2층 이상, 옥상, 언덕.
사다리를 올린 후 회수하면 접근 자체를 차단할 수 있다.

### 원칙 5 — 전기가 없는 세상을 상정하라
전기가 끊기면 현대 문명의 90%가 멈춘다.
**전기 없이 작동하는 것만이 진짜 생존 도구:**

| 도구 | 용도 | 전기? | 탄약? |
|------|------|-------|-------|
| 마체테/도끼 | 방어 + 도구 | ✗ | ✗ |
| 활/석궁 | 원거리 방어 | ✗ | 화살(재사용) |
| 정수 필터/끓이기 | 식수 | ✗ | 연료 |
| 수동 라디오 | 정보 | ✗(크랭크) | ✗ |

## PART 2: 시나리오별 특수 대응법

### 만약 "빠른 감염"이 온다면
> 참고 단서: 광견병의 공격성, 《부산행》《28 Days Later》의 상황 묘사

감염 후 수 분 내 변환. 감염자가 달리는 상황.

| 상황 | 대응 |
|------|------|
| 감염자 목격 | 즉시 도주. 교전 금지. 거리 확보 최우선 |
| 실내에 있다면 | 문 잠금 → 바리케이드 → 소리 최소화 |
| 차량 가능 | 시내 탈출. 교외/섬/산 방향 |
| 누군가 물렸다면 | 격리. 관찰. 변환 조짐 시 분리 |
| 어둠 | 감염자가 시각 의존형이라면 어둠이 은신처가 될 수도 |

**핵심**: 속도전. 판단과 행동이 빨라야 한다.

### 만약 "느린 감염"이 온다면
> 참고 단서: 동충하초의 실제 행동 조종, 《The Last of Us》의 단계별 변이

감염 후 수 일~수 주에 걸쳐 서서히 변화. 초기엔 정상으로 보임.

| 상황 | 대응 |
|------|------|
| 주변인 행동 이상 | 미세한 변화 관찰 — 식습관, 수면, 공격성 |
| 오염 지역 의심 | 마스크/방독면 필수. 피부 노출 최소화 |
| 오염 식수/식품 | 반드시 끓여서. 출처 불명 음식 금지 |
| 감염 초기 본인 | 자가 격리. 동료에게 알림. 기록 남기기 |

**핵심**: 조기 발견이 생사를 가른다. "괜찮아 보인다"가 가장 위험.

### 만약 "화학 사고"가 난다면
> 참고 단서: 보팔 사고, 후쿠시마, 《Dark Hole》의 연기 묘사

특정 지역에서 시작. 바람/물을 따라 확산.

| 상황 | 대응 |
|------|------|
| 이상한 냄새/연기 | 즉시 풍향 반대 방향 대피 |
| 실내 | 창문 닫기, 환기구 테이프 밀봉, 높은 층 |
| 피부 노출 | 흐르는 물 15분 이상 세척 |
| 오염 지역 통과 | 방독면 + 비닐 + 장갑 + 장화 |

**핵심**: 풍향 = 생사. 바람 반대쪽으로 움직여라.

### 만약 "사회 붕괴"가 온다면
> 참고 단서: 카트리나, 코로나 초기, 텍사스 대정전 — 좀비 없이도 오는 시나리오

전력 두절, 통신 마비, 치안 공백, 물류 중단.

| 상황 | 대응 |
|------|------|
| 정전 | 손전등, 양초, 수동 라디오 |
| 단수 | 욕조에 물, 빗물 수집, 정수 |
| 식량 부족 | 비상식량 3일분 상비 |
| 치안 공백 | 이웃과 연대. 야간 경비 |
| 통신 두절 | 가족 집결 장소 사전 약속 |

## PART 3: 전통 지혜에서 뽑은 대응 힌트

수천 년간 인류가 "이해할 수 없는 위협"에 대응해온 방법들.

### 불(火) — 거의 모든 문화의 핵심 방어 수단
북유럽은 드라우그르를 화장했고, 중세 유럽은 리버넌트를 태웠고, 동양은 횃불로 방어했다.
**실전**: 불은 조명·난방·조리·신호·방어를 동시에 해결하는 최고의 도구.

### 소금 — 정화와 보존의 상징
아이티에서는 좀비에게 소금을 먹이면 자아 회복. 다른 문화에서도 정화 도구.
**실전**: 식품 보존, 상처 소독(생리식염수), 다용도.

### 높은 곳 / 물(水) — 자연의 방벽
대부분의 전설에서 언데드는 수영 못 하고, 높은 곳을 오르기 어렵다.
**실전**: 섬, 높은 건물, 다리 끊기 — 영흥도가 완벽한 이유.

### 빛 / 새벽 — 어둠의 끝
강시는 닭 울음에 멈추고, 킹덤의 역병은 햇빛에 동면한다.
**실전**: 야간이 가장 위험하다는 가정. 활동은 낮에.

## PART 4: 좀비파크 훈련으로 연결

### 기본 코스: "첫 24시간 생존"
1. 상황 인지 브리핑 → 2. 생존 배낭 꾸리기 → 3. 바리케이드 구축 → 4. 정수 실습 → 5. 무소음 통신법

### 중급 코스: "3일 버티기"
1. 야간 이동 → 2. 오염 지대 통과 → 3. 식량 확보 → 4. 부상자 처치 → 5. 팀 의사결정

### 고급 코스: "거점 방어"
1. 다리 봉쇄 작전 → 2. 정찰 시스템 → 3. 자급자족 → 4. 장기 생존 → 5. 복합 위기 시나리오
