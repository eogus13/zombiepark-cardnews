## 프로젝트 한 줄 요약
영흥도(인천 옹진군)에 **실제 아포칼립스 대비 생존 거점 겸 테마파크**를 만드는 프로젝트.
디즈니랜드급 몰입감 + 실전 서바이벌 교육 + 리조트 복합단지.

## 핵심 원칙 (절대 변하지 않는 것)
1. **"좀비파크는 테마파크가 아니라, 진짜로 작동하는 생존 거점이다"**
2. 모든 시설은 **이중 목적** — 평시에는 체험 어트랙션, 비상시에는 실전 인프라
3. **철저한 컨셉 유지** — 놀이기구도 실제 상황에서 쓸 수 있는 것만 (예: 자전거 페달 케이블카)
4. **전기가 끊긴 세상에서도 작동하는가?** 가 모든 설계의 검증 기준
5. 처음부터 대규모가 아닌, **작은 인프라부터 시작해서 인지도 쌓으며 확장**

## 현재 단계
- **순수 아이디어 단계** (2026년 4월~)
- 사업자등록 전, 투자/지원 수령 전
- 대화를 통해 컨셉과 계획을 구체화하는 중

## 확정된 결정 사항

### 입지
- **영흥도** (인천광역시 옹진군 영흥면, 면적 23.46km²)
- 영흥대교 **하나만 차단하면 완전 격리**되는 천연 요새
- 옹진군은 **인구감소지역 지정** → 각종 지원사업 우대

### 마스터플랜 존 배치
| 방위 | 구역명 | 기능 |
|---|---|---|
| 동쪽 (영흥대교) | 방어 전선 | 검문소, 바리케이드, **1단계 서바이벌 캠프** |
| 중앙 (국사봉) | 지휘본부 | 정찰 거점, 자전거 페달 케이블카, 통신 거점 |
| 서쪽 (장경리) | VIP존 | 숙박형 서바이벌 체험, 곤충 사육(니오타니 연계 가능) |
| 남쪽 (외리) | 에너지 구역 | 대안 에너지 (태양광/풍력/인력발전) |
| 남쪽 해안 (용담리) | 식량 채취 | 갯벌 해산물 채취 훈련 |
| 북쪽 (십리포) | 의료/후방 기지 | 의료 거점, 빗물 정수 시설 |

### 1단계 시작 시설 (우선순위 순)
1. **서바이벌 훈련 캠프** (3,000만~5,000만) — 영흥대교 인근, 텐트/장비 기반
2. **서바이벌 책방 "지식 보관소"** (1억~2억) — 100~150평, 실내+야외, "마지막 생존지의 지식 아카이브" 컨셉
3. **야외 원형 공연장 "생존자 집결지"** (2,000만~5,000만) — 책방 근처 별도 공간, 최종 목표는 격납고형 실내 공연장

### 수익 모델
- **자격증 등급 시스템**: 등급이 올라갈수록 접근 가능 영역 확장 → 반복 방문 유도
- **VIP 카드**: 숙박형 서바이벌 체험 (세미 하드코어, 72시간 생존 시뮬레이션)
- 도서/굿즈/카페/프로그램 참가비/기업 B2B 훈련

### 자금 확보 전략
- **투자보다 정부 지원사업이 먼저** (아이디어 단계에서 투자 유치는 비현실적)
- 주요 지원사업: 인천 관광스타트업(~3,800만), 관광벤처사업(~7,000만), 예비창업패키지(~1억)
- 상세 내용은 [[04. 지원사업 정리]] 참고

### 식량 자급 3축
| 축 | 내용 | 기반 |
|---|---|---|
| 해산물 | 갯벌 채취 (바지락, 굴) | 38.6km² 갯벌 |
| 작물 | 농경지 텃밭 재배 | 기존 농경지 |
| 곤충(단백질) | 곤충 사육 | 니오타니 곤충 사육관 (장경리, 운영자와 친분) |

## 알아둬야 할 사람/시설
- **니오타니 곤충 사육관**: 장경리 소재, 운영자와 친분 있음. 현재 정보습득 단계 (협업 논의 전). 마스터플랜에서 식량 인프라 한 축으로 배치
